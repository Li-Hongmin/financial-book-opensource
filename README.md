# 跟太太一起理财

> 献给每一对想要一起变得更踏实的伴侣

一本温暖的夫妻理财入门书，不是教你如何暴富，而是帮助你们找到「我们的理财方式」。

## 📖 在线阅读

https://li-hongmin.github.io/financial-book-opensource/

## ✨ 这本书的特点

- 💑 **夫妻视角**：理解彼此对钱的不同态度
- 🤝 **平等对话**：不说教，而是一起探索
- 📝 **真实场景**：基于真实对话和故事
- 🎯 **可执行**：每章都有小任务

## 🚀 本地构建

```bash
# 安装 mdBook
cargo install mdbook

# 构建
mdbook build

# 本地预览
mdbook serve
```
