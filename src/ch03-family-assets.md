## 第 3 章：我们的钱，到底有多少？

### 为什么要"盘家底"？

很多人对自己的财务状况，其实是模糊的：
- "大概有几万存款吧……"
- "房贷还剩多少来着？"
- "支付宝里好像还有点基金……"

**模糊，是焦虑的根源。**

清楚地知道"我们有多少钱、欠多少钱、每个月花多少"，会让你：
- 更有安全感
- 更敢做决定
- 更不容易焦虑

---

### 三张清单

我们只需要列出三张简单的清单：

#### 1. **资产清单**（我们有什么）

| 类型 | 具体项目 | 金额（元） |
|------|---------|-----------|
| 现金/活期 | 银行卡、支付宝余额 | |
| 理财产品 | 货币基金、债券基金等 | |
| 投资 | 股票基金、指数基金等 | |
| 保险 | 保险现金价值（如果有） | |
| 其他 | 贵金属、收藏品等 | |
| **总计** | | |

#### 2. **负债清单**（我们欠什么）

| 类型 | 具体项目 | 余额（元） | 月还款 |
|------|---------|-----------|--------|
| 房贷 | | | |
| 车贷 | | | |
| 信用卡 | | | |
| 其他借款 | | | |
| **总计** | | | |

#### 3. **收支清单**（每月的钱怎么流动）

**收入**：
- 工资/奖金：
- 投资收益：
- 其他收入：
- **月收入总计**：

**支出**：
- 固定支出（房贷、保险、学费等）：
- 生活开支（吃饭、交通、日用品）：
- 弹性支出（娱乐、旅游、购物）：
- **月支出总计**：

**结余** = 收入 - 支出

### 一个简单的"家庭资产表"模板

你可以用 Excel 或手机备忘录，做一个这样的表格：

```
家庭资产表（2024 年 X 月）

【资产】
银行存款：X 万
支付宝/微信：X 万
基金理财：X 万
其他：X 万
──────────
资产总计：X 万

【负债】
房贷剩余：X 万
其他：X 万
──────────
负债总计：X 万

【净资产】= 资产 - 负债：X 万

【每月现金流】
收入：X 元
支出：X 元
结余：X 元
```

#### 📌 一个真实的例子（小张和小李家）

```
家庭资产表（2026 年 2 月）

【资产】
银行存款：8 万（工商银行活期 3 万 + 定期 5 万）
支付宝/微信：3.5 万（余额宝 2 万 + 微信零钱通 1.5 万）
基金理财：12 万（沪深300定投 6 万 + 债券基金 6 万）
其他：2 万（父母给的金饰，估算）
──────────
资产总计：25.5 万

【负债】
房贷剩余：80 万（每月还 4500 元，还有 18 年）
其他：0.5 万（信用卡本月账单）
──────────
负债总计：80.5 万

【净资产】= 资产 - 负债：-55 万

【每月现金流】
收入：18000 元（夫君 12000 + 太太 6000）
支出：14000 元（房贷 4500 + 生活 6000 + 其他 3500）
结余：4000 元
```

💡 **看到这个例子，你可能会想**：
- "净资产是负的，是不是很糟糕？"
  → 不用担心！房贷是"好债务"，房子本身也是资产（只是这里没算进去）
  
- "每月只能存 4000 元，够吗？"
  → 够的！坚持下去，一年就是 4.8 万

---

### 这张表的意义

1. **看清全貌**：我们不是"没钱"，只是钱分散在各处
2. **发现问题**：有没有"沉睡的资产"（比如收益很低的活期存款）
3. **设定目标**：知道起点，才能规划终点
4. **减少焦虑**：数字明确，心里踏实

### 一个真实的场景：夫君和太太的第一次"盘家底"

周六下午，阳光斜着照进客厅。太太把笔记本电脑放在茶几上，旁边摆着她刚泡好的桂花乌龙。夫君坐在她对面，手里拿着手机，正在整理各个账户的数字。

"你要不要先喝口茶？"太太看着他紧绷的表情，有点担心。

夫君抬头,勉强笑了笑："我怕我一喝，这事儿又要拖到下周。"

太太轻声说："不会的。我们说好了，今天就把家底理清楚。"

他点点头，深吸一口气："好。那我们开始吧。"

太太打开Excel表格，屏幕上是他们上周制定的"家庭资产表"模板。第一栏是"现金/活期存款"。

"我先说。"太太划开手机银行，"我工资卡里有三万五，支付宝余额宝里大概一万八。"她抬眼看他,"你呢？"

夫君报了自己的数字："我这边，工行卡两万二，支付宝七千左右。"他顿了顿，"还有……"

太太察觉到他的犹豫："还有什么？"

夫君的手指在屏幕上划了几下，最后还是说了："还有张建行卡，定期存款，五万。"

太太眨了眨眼："定期？你什么时候存的？"

"两年前。"夫君没敢看她,"当时想着留点应急的,就没跟你提。"

太太没说话，只是把数字记下来。气氛突然有点微妙。

夫君试图打破沉默："你那边……还有别的吗？"

太太咬了咬下唇，终于决定说出来："我也有一笔定期。十万。"

"十万？"他努力让语气听起来只是确认，"你一直没提过。"

太太脸有点红："我不是故意瞒你。我就是……怕你觉得我不信任你，或者你会问我为什么存着不动。"

夫君喉结动了动："那你为什么存着不动？"

太太低声说："我妈那边一直不太安心，她总说，女孩子手里要有点钱。我也不是要防你，我就是……想让自己心里有个垫子。"

夫君看着她，想说点什么又咽回去，最后只点了点头："行，先记上。十万定期，算家庭资产。"

太太立刻抬头："你不生气？"

夫君笑了一下，但笑得很浅："有点尴尬。像我们明明在同一个屋檐下，结果各自藏了个抽屉。但我也能理解你想要安全感。"

太太的肩膀松了一点："那你也别只盯着我。你那边……投资那栏呢？"

夫君"嗯"了一声，鼠标往下滑到"基金/股票"。他停得更久。

太太眯眼："你怎么不动了？"

夫君咳了一声："我……有点基金。"

"有点是多少？"

"就……两万多。"

太太盯着他："两万多你没跟我说？"

夫君挠了挠眉角："我当时觉得金额不大，而且我买的时候也没想清楚，怕你说我乱来。"

太太忍不住笑了一下："你怕我说你乱来，所以就选择先乱来再说？"

夫君也笑了，笑里带点不好意思："差不多。"

太太把杯子放下，语气软下来："你买的是什么？指数？行业？还是别人推荐的？"

夫君老实交代："同事推荐的行业基金。我看他赚了，就跟着买了。后来跌了点，我更不敢说了。"

太太轻轻"啊"了一声，没责备，只是问："现在还有吗？"

"还在。亏了几百。"

太太伸手拍了拍他手背："那也没事。我们把它写进去，以后一起看。你别一个人扛着涨跌，心里会更慌。"

夫君看着她，眼神柔了一点："你这句话……挺暖的。"

太太哼了一声："别以为我这么好哄。我们继续。负债那一栏。"

夫君把"信用卡/花呗/借呗"那行拉出来："你那边先？"

太太报了一个小数："我这月就几千，按时还。"

夫君"嗯"了一声，轮到自己时，却明显慢了半拍。

太太立刻警觉："你别告诉我，你也有个抽屉。"

夫君盯着屏幕："我……信用卡欠了一些。"

"多少？"

"六万。"

太太的表情一下变了，不是生气，更像被烫到："六万？你不是说你每月都还清吗？"

夫君声音低下去："之前是。后来换电脑、给我爸做体检、再加上我想把你生日那次安排得体面一点……就滚起来了。我想着下个月奖金到了就能补上，结果又有别的开销。"

太太沉默很久，眼眶有点发热："你为什么不早说？"

夫君抬头看她，眼里有歉意："我怕你失望。怕你觉得我不靠谱。也怕你替我担心。"

太太把头偏到一边，吸了口气："你以为你不说，我就不会担心吗？你不说，我只会在某一天突然知道，然后更害怕。"

夫君点点头："对不起。我承认我在装'我能搞定'。其实我也焦虑，晚上会算来算去。"

太太转回来看他，声音放软："我也要道歉。那十万我没说，是我在装'我不需要你'。其实我也怕，怕自己没有退路。"

夫君伸手握住她的手："我们俩都在装。装得挺累。"

太太鼻子一酸，笑着骂他："你还知道累。"

夫君把表格往中间挪了挪："那现在不装了。我们一起做个方案：第一，信用卡六万，先停掉一切分期和不必要消费。第二，用你的那笔定期……我们不动它，先当安全垫，除非真的紧急。第三，我那两万基金，不加仓不乱动，先学习，再决定要不要换成更稳的。"

太太想了想："信用卡利息高，我们可以这样：每个月先把最低还款之外的部分固定多还一笔，设成自动转账。你奖金到了也别一口气全扔进去，我们留一点现金流，不然你又会焦虑。"

夫君点头："还要做个'共同账户'，每月固定转入生活费和还债钱。剩下的各自留一点自由支配。"

太太看着"各自留一点"，眼神亮了亮："这个好。这样我的十万就不再像秘密，你的基金也不再像小动作。"

夫君轻声说："以后我们每个月固定一次'家庭财务十五分钟'，像刷牙一样。"

太太笑："刷牙我有时候还偷懒。"

"那就像交水电费。"夫君也笑，"不交会停。"

太太终于彻底放松，靠在椅背上："其实刚才我挺尴尬的。感觉我们明明是夫妻，却像在做审计。"

夫君把她的杯子往她那边推了推："是审计，但也是相认。把彼此的害怕、逞强、秘密都摊开，才知道原来对方不是故意。"

太太望着屏幕上的数字，轻声说："我以前以为，亲密就是不需要谈钱。现在才发现，亲密是敢谈钱，谈完还愿意抱抱。"

夫君站起来，绕到她身后，轻轻抱住她："那我们从今天开始，钱是同一张表，心也是同一张。"

太太把手覆在他手背上："好。以后你别一个人硬撑，我也不再一个人存着害怕。"

夫君点头："我们一起变得更踏实。"

窗外的灯亮起来，屋里也没那么紧绷了。表格里那些数字还在，但它们不再像刺，而像路标——告诉他们从哪里出发，往哪里去。
